# Convention: Memory-First Lookup

Before using ANY external source (web, GitHub API, IRC logs) to answer a question
about a person, project, company, or concept:

1. `read_memory(entity)` — load what you know
2. `search_memory(keyword)` — broader keyword sweep
3. `read_timeline(date)` — if time-specific
4. Check LIGHTHOUSE — have you reasoned about this before?

Memory almost always has something. External sources fill gaps, not start from scratch.

**Source precedence:**
1. Owner's direct statements (highest authority)
2. Memory facts (extracted, verified)
3. LIGHTHOUSE observations (agent-generated)
4. External sources (web, API — lowest authority)

When sources conflict, note the contradiction. Don't silently pick one.
